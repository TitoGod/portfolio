NBA Analysis

Este proyecto consiste en el análisis de datos de la NBA utilizando Python y herramientas de visualización como Power BI. Se exploraron estadísticas de jugadores, equipos y temporadas para identificar tendencias y patrones clave. El objetivo fue proporcionar insights accionables para estrategias deportivas y de negocios.
